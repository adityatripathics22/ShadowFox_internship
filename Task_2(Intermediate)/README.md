# Whistle Podu Central

An immersive IPL team website concept for Chennai Super Kings.

## Features

- Responsive hero section with countdown to the next featured match
- Team story and franchise identity overview
- Squad section with role-based player filtering
- Upcoming fixtures table
- Team stats cards
- News update cards
- Fan zone with a simple poll, social hashtag chips, and live update feed

## Project Files

- `index.html` - page structure and content
- `styles.css` - visual design, layout, and responsive behavior
- `script.js` - mobile menu, player filtering, fan poll, and countdown logic

## Run

This is a static project, so you can open `index.html` directly in a browser.

If you prefer a local server, run one of these commands inside the project folder:

```powershell
python -m http.server 8000
```

or

```powershell
npx serve .
```
